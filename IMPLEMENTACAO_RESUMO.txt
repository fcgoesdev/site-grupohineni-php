1. VERIFICAÇÃO LOCAL:
   - Instalar Apache + PHP localmente
   - Testar cada página no browser
   - Testar formulário de contato
   - Testar filtros de clientes
   - Verificar responsividade

2. PREPARAÇÃO DE SERVIDOR:
   - Escolher hosting (recomenda-se Apache + PHP 7.4+)
   - Configurar domínio
   - Habilitar HTTPS (SSL)
   - Testar email (SMTP ou mail())

3. DEPLOYMENT:
   - Fazer upload via FTP/SFTP/Git
   - Ajustar permissões de arquivos
   - Verificar .htaccess
   - Teste de acesso (HTTP 200)

4. TESTES PRÉ-PRODUÇÃO:
   - Testar em múltiplos browsers
   - Testar em múltiplos dispositivos
   - Teste de performance
   - Teste de formulário (email real)
   - Verificar logs de erro

5. MONITORAMENTO PÓS-DEPLOY:
   - Coletar feedback de usuarios
   - Monitorar logs
   - Fazer ajustes necessários
   - Backup automático
